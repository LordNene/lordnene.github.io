Personal Learning Environment

For a university subject "Information Literacy Course" we were tasked to create a map of our learning environment in Miro. It is an interesting thought experiment for IT field, where I feel like a lot of apps overlap in their use cases - such as Github which can be used for a lot of these areas (with its Project Boards, Pull requests, Issue tracking etc). But even with this overlap, it quickly became apparent how I don't know and use much in terms of feedback. It's something to focus on in the future, but for now the contrast between other areas and paltry 2 stickers under "Feedback" is a thought provoking result.
